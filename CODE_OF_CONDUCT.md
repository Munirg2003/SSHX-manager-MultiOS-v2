# Contributor Covenant Code of Conduct

Our community is dedicated to providing an open and welcoming experience. All contributors are expected to adhere to the Contributor Covenant Code of Conduct.

For the full text, see: https://www.contributor-covenant.org/version/2/0/code_of_conduct/

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported to the project maintainers.
